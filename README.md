# サンプルコードの実行環境

検証した環境は下記の通りです

- Ruby 2.3.3
- macOS Siera

## 起動方法

- middleman を利用しています
- このリポジトリをcloneして利用してください

```
bundle install
bundle exec middleman server
open http://localhost:4567
```
